# Todo List App

A simple and clean todo list application built with React and Tailwind CSS.


## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Filter by **All**, **Active**, or **Completed**
- Clear all completed tasks at once

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

No installation needed. Just follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/maaz-afzal/todo-app.git
   ```

2. Go into the project folder:
   ```
   cd todo-app
   ```

3. Open `index.html` in your browser and you are good to go.
